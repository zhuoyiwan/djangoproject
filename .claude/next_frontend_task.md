# Next frontend session task

Read this file first in unattended frontend sessions.

## Task

Start by reading the current frontend routes, shared styles, and the backend API docs that the touched page depends on. Continue with the next smallest complete frontend increment after the current repo state.

## Visual direction

- 只做前端
- 页面文案优先中文
- 整体视觉必须是高级、现代、克制的玻璃拟态 UI
- 参考苹果 Human Interface Guidelines 的空间层级、留白和内容组织
- 使用连续圆角 squircle、半透明磨砂玻璃、轻微白色发光内边框、柔和多层环境阴影
- 使用干净中性底色，叠加模糊、流动、鲜艳但不过度嘈杂的抽象渐变背景
- 使用高对比无衬线几何字体
- 不要做模板化后台，不要做廉价炫技，不要做拥挤布局

## Constraints

- Backend only as dependency reference; never edit backend code
- Keep changes small but complete
- Run frontend build validation for touched work
- Stop after one coherent increment
